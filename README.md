# svelte

## dev env

- requirement "bun 1.0.7"

## how to install app

1. first you clone this repo

```zsh
    git clone <repo url>
```

2. If your computer not installed Bun, please install first.

3. If you have Bun. You put into Terminal or CLI tool below command.

```zsh
bun install
```

4. And...

```zsh
bun dev
```

5. Happy Svelte.
